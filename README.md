[로그인하고 확인하기] 버튼 클릭 시 로그인 페이지로 강제 이동.3. 하단 네비게이션 구조 (Bottom Tab Bar)탭(Tab) 명칭연결 화면주요 기능 및 목적🏠 홈Main Dashboard핫한 꿈 큐레이션 노출 / 오늘의 꿈 기록 CTA💬 커뮤니티Community Feed익명 기반 유저 간 해몽 요청 및 소통 공간➕ 꿈 기록Write Form (Full)클릭 시 새 페이지(Full-page)로 이동하여 작성📖 일기장My Dream Diary내가 기록한 꿈 목록 모아보기 (캘린더/리스트)👤 마이페이지My Profile내 등급, 뱃지, 프로필 꾸미기 및 설정4. 핵심 화면별 상세 명세 (Screen UI & Flow)4.1. 메인 홈 (Home)[상단] 지금 핫한 꿈글! (캐러셀 슬라이드):최근 72시간 내 핫한 게시글 노출. (초기 Cold Start 시 관리자 작성 '유명 꿈 썰' 더미 데이터 고정 노출).[중단] 오늘 나의 꿈은? (Floating CTA):즉시 작성 페이지로 넘어가는 직관적인 입력 유도 버튼.[하단] 최근 내 꿈 요약:최근 기록한 꿈 키워드 1~2개 또는 미확인 해몽 결과 리스트업.4.2. 꿈 작성 페이지 (Write Form)기본 정책: 작성 직후 기본값은 '나만 보기(비공개)'. 커뮤니티 공유는 해몽 결과 확인 후 유저가 선택함.입력 항목:제목 (필수): Text Input (최대 30자 제한).기분 태그 (필수): Button/Chip 형태 (예: 😃상쾌함, 😱무서움 등) / 다중 선택 지원.꿈 내용 (필수): Text Area (10자 이상 ~ 1000자 제한).키워드 태그 (자동): 유저가 텍스트 입력 시, AI가 핵심 사물/인물(예: #뱀, #가족)을 추출해 자동 해시태그 부여.작성 이탈 방지: 웹 브라우저 뒤로 가기 및 스와이프 시 "작성 중인 내용이 사라집니다. 나가시겠습니까?" 모달 팝업 필수 노출.4.3. 해몽 결과 및 공유 (Result & Share)[저장] 클릭 후 로딩: 3~5초간 AI 분석 Lottie 애니메이션 노출 (이탈 방지).결과 노출 화면:기본 해몽 텍스트 (무료).[광고 보고 심층 해몽 보기] 버튼 노출.[커뮤니티에 이 꿈 공유하기] 버튼 노출 → 클릭 시 익명 커뮤니티 피드로 퍼블리싱.4.4. 커뮤니티 & 게이미피케이션 (Community)게시글 상세: 익명 닉네임 노출, 해몽 댓글 작성 기능, '길몽' 공감 투표 기능.활동 보상 시스템 (Phase 1):댓글 작성 수, 질문자의 '채택' 수에 따라 유저 등급 상승.특정 등급 도달 시 [전문 해몽가 뱃지] 등 프로필 꾸미기 권한 부여.5. 🚩 기획 누락 포인트 및 운영 방안 (Risk Management)모바일 웹 접근성(PWA) 대응:앱이 아니므로 재방문율(Retention)이 낮을 수 있습니다. 랜딩 시 팝업으로 "홈 화면에 바로가기 아이콘을 추가해 보세요!" 라는 가이드를 노출해야 합니다.로그인 후 리다이렉트(Redirect):비회원이 특정 '핫한 글'을 보려고 클릭해 로그인을 마쳤다면, 메인 홈이 아니라 원래 보려던 그 게시글 상세 페이지로 즉시 이동(Deep Link) 시켜야 유저 경험이 단절되지 않습니다.어뷰징 필터링 정책:활동 뱃지를 얻기 위한 무의미한 댓글 도배(예: "ㅋㅋ", "좋은 꿈이네요")를 방지하기 위해, 댓글 글자 수 하한선(예: 10자 이상) 제약이 필요합니다.
